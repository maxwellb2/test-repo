## What This Repo Is

This repository is a personal workshop for trying things out before they deserve a dedicated home.

It is where small implementations, feature spikes, language experiments, library evaluations, performance demos, and "I wonder if this works..." ideas can live while they are still taking shape. Once an idea becomes substantial, reusable, or worth presenting as its own thing, it can graduate into a focused repository with cleaner history, sharper documentation, and a purpose-built structure.

> Think of this repo as the launchpad, not the final spacecraft.

## What Belongs Here

```text
personal-projects/
|
|-- language experiments
|-- library testbeds
|-- performance sandboxes
|-- feature prototypes
|-- proof-of-concept apps
|-- small demos
`-- ideas that are still earning their name
```

This repo is intentionally broad. A folder might be a one-file demo, a weekend experiment, a benchmark, a UI prototype, or a full project seed. The common thread is experimentation.

## Guiding Principle

<div align="center">

### Prototype here. Prove it here. Polish it elsewhere.

</div>

This repository keeps early exploration fast and lightweight. It is allowed to be playful, uneven, and experimental because that is the point. The best ideas can leave this lab with lessons learned, sharper requirements, and a clearer reason to exist.
